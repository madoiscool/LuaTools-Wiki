## Common Solutions

### Quick Fixes

1. **Verify Game Files** (Steam)
   - Right-click game → Properties → Local Files → Verify integrity

2. **Update Drivers**
   - Graphics drivers
   - Network drivers
   - System drivers

3. **Clear Cache**
   - Steam download cache
   - Game cache files
   - Browser cache (if applicable)

4. **Run as Administrator**
   - Right-click game executable
   - Select "Run as administrator"
